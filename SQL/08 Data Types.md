**Common Data Types**
- `VARCHAR(n)` : Variable-length string, up to n characters
    
- `INT` : Integer numbers
    
- `DATE` : Calendar date
    
- `BOOLEAN` : true/false values
    
- `FLOAT`, `NUMERIC` : Decimal numbers
    
- `SERIAL` : Auto-increment (Postgres)
    
- `INT AUTO_INCREMENT` : Auto-increment (MySQL)
